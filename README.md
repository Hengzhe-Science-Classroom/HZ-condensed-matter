Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 5 · Research Frontier

## What You'll Learn

- Crystal structure and reciprocal lattice
- Phonons and thermal properties
- Free electron model
- Band theory and semiconductors
- Magnetism
- Superconductivity

- Charles Kittel, *Introduction to Solid State Physics* (8th ed., Wiley)
- Steven H. Simon, *The Oxford Solid State Basics* (Oxford)
